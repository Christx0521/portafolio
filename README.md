# Portafolio — Christian Xavier Nieto

Portafolio personal de desarrollador junior full-stack y game dev. Construido con
Vite + React + TypeScript y CSS Modules, sin dependencias de estilos externas.

## Stack

- Vite + React 19
- TypeScript (`strict: true`)
- CSS Modules (tema oscuro)

## Secciones

- Hero
- Sobre mí
- Proyectos (NEO, Pac-Man, BananaMovement, Torre de Hanói, App PHP)
- Skills & Stack
- Contacto

## Scripts

```bash
npm install      # instalar dependencias
npm run dev      # servidor de desarrollo
npm run build    # type-check + build de producción
npm run preview  # previsualizar el build
```

## Personalizar

- Proyectos y datos de contacto: `src/data/projects.ts`
- Tema y variables de color: `src/index.css`
